# DeathNote Mbaretech2 - Índice General

Bienvenido a la carpeta de conocimiento y bitácora del proyecto Megasumo Mbaretech2. 
Esta carpeta está diseñada para mantener el contexto del proyecto vivo, registrar descubrimientos de hardware, y establecer reglas estrictas para el desarrollo futuro.

## Archivos en esta carpeta:

1. **[01_Reglas_IA.md](./01_Reglas_IA.md)**
   * **Propósito:** Lectura OBLIGATORIA para cualquier agente de Inteligencia Artificial. Contiene restricciones de programación y la directiva obligatoria de registrar acciones.

2. **[02_Estado_Actual.md](./02_Estado_Actual.md)**
   * **Propósito:** Fotografía del estado físico y lógico del robot. Documenta bugs de hardware conocidos (como el cruce de pines) y cómo funciona la máquina de estados actual.

3. **[03_Bitacora_Diaria.md](./03_Bitacora_Diaria.md)**
   * **Propósito:** Registro de logros, razonamientos complejos y decisiones estratégicas importantes a nivel proyecto.

4. **[04_Registro_Acciones_RAW.md](./04_Registro_Acciones_RAW.md)**
   * **Propósito:** Historial detallado de cambios menores de código, alimentado automáticamente por la IA tras cada modificación para mantener trazabilidad.

5. **[99_Razonamientos_y_mejoras.md](./99_Razonamientos_y_mejoras.md)**
   * **Propósito:** Bitácora de ideas estratégicas, análisis de debilidades del código actual (ej. trampas de banderas enemigas) y planes de implementación a futuro.
